<h1>–<h6>: Os elementos HTML de cabeçalho.

(Voltar)

As Tags Heading que são de H1 ate H6, são geralmente utilizadas como subtítulos de uma página, que é do maior para o menor.
Exemplo:

Buffet Infantil da Carochinha - <h1>

Festas temáticas - <h2>

Festas do Barney - <h3>

Festas dos Angry Birds - <h4>

Festa da galinha pintadinha - <h5>
Ponto de referência - <h6>

<p> O elemento Parágrafo.

(Voltar)

O elemento HTML <p> representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco. Já em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.
Exemplo:

Este é o segundo parágrafo do texto.

Elemento HMTL <mark>.

(Voltar)

Representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular. Por Exemplo, pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque.
Exemplo:

O elemento <mark> é usado para destacar partes do texto

O elemento de comentário lateral <small>.

(Voltar)

O elemento HTML representa comentários laterais e letras pequenas, como direitos autorais e texto legal, independentemente de sua apresentação estilizada. Por padrão, ele renderiza o texto em um tamanho de fonte menor, como de para. <small> smallx-small
Exemplo:

Esta é a primeira frase. Esta frase inteira está em letras minúsculas.

O elemento Texto Idiomático <i>.

(Voltar)

O elemento HTML representa um intervalo de texto que se destaca do texto normal por algum motivo, como texto idiomático, termos técnicos, designações taxonômicas, entre outros. Historicamente, eles foram apresentados usando o tipo itálico, que é a fonte original da nomenclatura desse elemento. <i>.
Exemplo:

A frase latina Veni, vidi, vici é frequentemente mencionada na música, arte e literatura.

O elemento de anotação não articulada (sublinhado) <u>.

(Voltar)

O elemento HTML representa um intervalo de texto embutido que deve ser renderizado de uma forma que indique que possui uma anotação não textual. Isso é renderizado por padrão como um simples sublinhado sólido, mas pode ser alterado usando CSS. <u>.
Exemplo:

Este parágrafo inclui uma palavra com grafia erroneamente.

O elemento de forte importância <strong>.

(Voltar)

O elemento HTML indica que seu conteúdo tem forte importância, seriedade ou urgência. Os navegadores normalmente renderizam o conteúdo em negrito. <strong>.
Exemplo:

Importante: antes de continuar, certifique-se de adicionar bastante manteiga.

O elemento Lista Ordenada <ol>.

(Voltar)

O elemento HTML representa uma lista ordenada de itens — normalmente renderizada como uma lista numerada. <ol>.
Exemplo:

  1. Introdução
  2. Lista de Reclamações
  3. Conclusão

O elemento Lista Não Ordenada <ul>.

(Voltar)

O elemento HTML representa uma lista não ordenada de itens, normalmente renderizada como uma lista com marcadores. <ul>.
Exemplo:

  • Primeiro item
  • Segundo item
  • Terceiro item

O elemento Item de lista <li>.

(Voltar)

O elemento HTML é usado para representar um item em uma lista. Ele deve estar contido em um elemento pai: uma lista ordenada ( ), uma lista não ordenada ( ) ou um menu ( ). Em menus e listas não ordenadas, os itens de lista geralmente são exibidos usando marcadores. Em listas ordenadas, eles geralmente são exibidos com um contador crescente à esquerda, como um número ou uma letra. <li>.
Exemplo:

  1. Terceiro item
  2. Quarto item
  3. Quinto item

O elemento de hiperligação <a>.

(Voltar)

O elemento <a> em HTML (ou elemento âncora), com o atributo (href) cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.
Exemplo:

  • Website
  • E-mail
  • Telefone

O elemento de linha de horizontal <hr>.

(Voltar)

O elemento HTML <hr> representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção). Nas versões anteriores do HTML, representava uma linha horizontal. Pode continuar sendo exibida como uma linha horizontal nos navegadores, mas agora está definida em termos semânticos, em vez de termos de apresentação.
Exemplo:

Este é o primeiro parágrafo do texto. Este é o primeiro parágrafo do texto.

Este é o segundo parágrafo do texto. Este é o segundo parágrafo do texto.

O elemento subscrito <sub>.

(Voltar)

O elemento HTML especifica o texto embutido que deve ser exibido como subscrito por motivos exclusivamente tipográficos. Os subscritos normalmente são renderizados com uma linha de base reduzida usando texto menor.<sub>
Exemplo:

*Números das notas de rodapé
De acordo com os cálculos de Nakamura, Johnson e Mason25 isso resultará na aniquilação completa de ambas as partículas.

**Subscritos variáveis
As posições das coordenadas horizontais ao longo do eixo X são representadas como x1 … xn.

***Fórmulas químicas
Quase a molécula favorita de todos os desenvolvedores é C8H10N4O2, que é comumente conhecido como "cafeína".

O elemento sobrescrito <sup>.

(Voltar)

O elemento HTML especifica o texto embutido que deve ser exibido como sobrescrito apenas por motivos tipográficos. Os sobrescritos geralmente são renderizados com uma linha de base elevada usando texto menor.<sup>
Exemplo:

*Expoentes
Uma das equações mais comuns em toda a física é E=mc2.

**Letras superiores
Robert apresentou seu relatório Para Bernarda.

***Números ordinais
O número ordinal "quinto" pode ser abreviado em vários idiomas da seguinte forma:

  • Inglês: 5th
  • Francês: 5ème

O elemento Block Quotation <blockquote>.

(Voltar)

O elemento HTML indica que o texto incluído é uma citação estendida. Normalmente, isso é renderizado visualmente por recuo (consulte Notas para saber como alterá-lo). Uma URL para a fonte da citação pode ser fornecida usando o atributo, enquanto uma representação de texto da fonte pode ser fornecida usando o elemento.<blockquote> cite<cite>
Exemplo:

A topologia de conexão é limitada a um único caminho ponto a ponto para cada transportadora, usado com transportadoras padrão, mas muitas transportadoras podem ser usadas sem interferência significativa entre si, fora do início da primavera. Isso se deve ao espaço de éter 3D disponível para os portadores, em contraste ao éter 1D usado por IEEE802.3. Os portadores têm uma colisão intrínseca sistema de prevenção, o que aumenta a disponibilidade.

O elemento de entrada (entrada de formulário) <input>.

O elemento é um dos mais poderosos e complexos em todo HTML devido ao grande número de combinações de tipos de entrada e atributos.

O elemento de texto excluído <del>.

(Voltar)

O elemento HTML representa um intervalo de texto que foi excluído de um documento. Isso pode ser usado ao renderizar "mudanças de faixa" ou informações de diferenças de código-fonte, por exemplo. O elemento pode ser usado para o propósito oposto: para indicar o texto que foi adicionado ao documento.<del> <ins>
Exemplo:

Minha cor favotita é azul vermelho.

O elemento de abreviação <abbr>.

(Voltar)

O elemento HTML representa uma abreviação ou acrônimo.<abbr>
Ao incluir uma abreviatura ou acrônimo, forneça uma expansão completa do termo em texto simples no primeiro uso, juntamente com o <abbr>símbolo para marcar a abreviatura. Isso informa ao usuário o que a abreviação ou sigla significa. O atributo opcional titlepode fornecer uma expansão para a abreviação ou acrônimo quando uma expansão completa não estiver presente. Isso fornece uma dica para os agentes de usuário sobre como anunciar/exibir o conteúdo enquanto informa a todos os usuários o que a abreviação significa. Se presente, titledeve conter esta descrição completa e nada mais.
Exemplo:

CSS é uma linguagem que descreve o estilo de um documento HTML.
